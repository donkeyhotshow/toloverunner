## ⚡ БЫСТРЫЙ СТАРТ

Откройте игру в браузере и выполните следующие тесты:

---

## 🎯 КРИТИЧЕСКИЕ ТЕСТЫ

### **1. Базовое Движение** (2 минуты)

- [ ] Запустить игру
- [ ] Переключить полосы (A/D) - плавное движение без рывков
- [ ] Прыжок (Space) - стабильная траектория
- [ ] Двойной прыжок - без глитчей
- [ ] Dash (Shift) - моментальная смена полосы

**Ожидаемое:** Все анимации плавные, нет телепортов, нет мерцания

---

### **2. Визуальные Эффекты** (2 минуты)

- [ ] Проверить линии движения (motion lines) - не мерцают
- [ ] Проверить след из частиц (particle trail) - плавный
- [ ] Проверить облака пыли при смене полосы - корректные
- [ ] Проверить анимацию хвоста - органичная, без разрывов

**Ожидаемое:** Все эффекты отображаются корректно, нет артефактов

---

### **3. Squash & Stretch** (1 минута)

- [ ] Быстро переключать полосы (spam A-D) - горизонтальное сжатие
- [ ] Прыгать - вертикальное растяжение
- [ ] Столкнуться с препятствием - эффект удара
- [ ] Активировать speed boost - вытянутая форма

**Ожидаемое:** Все деформации плавные, модель не коллапсирует

---

## 🔥 СТРЕСС-ТЕСТЫ

### **4. Низкий FPS** (3 минуты)

**Подготовка:**
1. Открыть DevTools (F12)
2. Performance → CPU: 6x slowdown
3. Запустить игру

**Тесты:**
- [ ] Движение остается плавным (с учетом замедления)
- [ ] Нет "прыжков" позиции игрока
- [ ] Эффекты не ломаются
- [ ] Коллизии работают корректно

**Ожидаемое:** Игра работает стабильно даже на низком FPS

---

### **5. Высокий FPS** (2 минуты)

**Подготовка:**
1. Включить мощную производительность
2. Закрыть другие приложения
3. Запустить игру на максимальных настройках

**Тесты:**
- [ ] Анимация хвоста не "дергается"
- [ ] Линии движения плавные
- [ ] След из частиц равномерный
- [ ] Физика стабильная

**Ожидаемое:** На высоком FPS нет артефактов или ускорения

---

### **6. Экстремальные Движения** (3 минуты)

- [ ] Быстрое переключение полос (A-D spam 10 сек)
- [ ] Комбинация прыжок + dash
- [ ] Прыжок + speed boost + dash
- [ ] Столкновение во время dash
- [ ] Столкновение во время прыжка

**Ожидаемое:** Игрок не "выходит" за границы, нет телепортов

---

## 🕐 ДЛИТЕЛЬНЫЕ ТЕСТЫ

### **7. Продолжительная Сессия** (10 минут)

- [ ] Играть непрерывно 10 минут
- [ ] Проверить стабильность анимаций
- [ ] Проверить FPS (не ухудшается)
- [ ] Проверить память (DevTools → Memory → нет утечек)
- [ ] Проверить консоль (нет ошибок)

**Ожидаемое:** Производительность стабильна на протяжении всей сессии

---

### **8. Тест Памяти** (15 минут)

**Подготовка:**
1. DevTools → Memory → Heap snapshot
2. Запустить игру

**Процесс:**
1. Сделать snapshot (начало)
2. Играть 5 минут
3. Сделать snapshot (середина)
4. Играть еще 5 минут
5. Сделать snapshot (конец)
6. Сравнить размеры

**Ожидаемое:** Память растет незначительно, нет больших утечек

---

## 📊 СПЕЦИФИЧНЫЕ ТЕСТЫ

### **9. Проверка Ограничений** (2 минуты)

**В DevTools Console выполнить:**

```javascript
// Проверка ограничения скейла
const player = document.querySelector('[data-player]');
// Искусственно установить экстремальный scale
// Система должна clamp его в [0.2, 3.0]

// Проверка ограничения ротации
// Должен быть в диапазоне [-π/3, +π/3]
```

**Вручную:**
- [ ] Проверить, что игрок не становится слишком маленьким
- [ ] Проверить, что игрок не становится огромным
- [ ] Проверить, что наклон при движении умеренный

---

### **10. Граничные Случаи** (3 минуты)

- [ ] Перейти на крайнюю левую полосу (-2)
- [ ] Перейти на крайнюю правую полосу (+2)
- [ ] Попробовать прыгнуть "за границы" (не должно быть возможности)
- [ ] Активировать все powerup одновременно
- [ ] Собрать powerup во время столкновения

**Ожидаемое:** Все граничные случаи обработаны корректно

---

## 🔍 ВАЛИДАЦИЯ CONSOLE

### **11. Проверка Предупреждений** (постоянно)

Во время всех тестов следить за консолью:

**Допустимые warning:**
- `[Player] Invalid target position detected` - редко (fail-safe)
- `[ToonSperm] Invalid wave values` - редко (fail-safe)
- `[HorizontalMotionLines] Invalid target position` - редко (fail-safe)

**Недопустимые:**
- ❌ `NaN detected`
- ❌ `Infinity detected`
- ❌ Любые JavaScript errors
- ❌ Частые warnings (>10/сек)
