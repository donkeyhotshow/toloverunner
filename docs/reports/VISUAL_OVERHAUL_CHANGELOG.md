#### 1. ✅ Прогрессивная Скорость
- **Файлы:** 
  - `store/gameSlice.ts` - логика прогрессии
  - `constants.ts` - RUN_SPEED_BASE
- **Что сделано:**
  - Начальная скорость: **30.0 → 10.0** (медленный старт)
  - MIN_SPEED = 10.0, MAX_SPEED = 25.0
  - Прогрессия: 2 минуты (120 сек) для достижения максимума
  - Формула: `lerp(MIN_SPEED, MAX_SPEED, progress)`
  - Плавный lerp для комфортного разгона
- **Эффект:** Игрок успевает понять управление, скорость растет постепенно

#### 2. ✅ Улучшенная Смена Лайнов  
- **Файл:** `core/physics/PlayerPhysicsLogic.ts`
- **Изменения:**
  - `laneSpeed`: 22 → **30** (жесткость пружины ~15, как в требованиях)
  - `laneDamping`: 0.75 → **0.80** (для контроля)
  - Spring-based физика с высокой скоростью отклика
  - Нет телепортации, нет медленного "плавания"
- **Эффект:** Резкий, отзывчивый переход между лайнами (как в Subway Surfers)

#### 3. ✅ Настройка Прыжка и Гравитации
- **Файлы:**
  - `core/physics/PlayerPhysicsLogic.ts`
  - `constants.ts`
- **Изменения:**
  - `gravity`: 140 → **35** (не лунная, не кирпичная - оптимальная 30-40)
  - `jumpForce`: 42 → **18** (пропорционально новой гравитации)
  - `doubleJumpForce`: 36 → **14**
  - GRAVITY_Y: 65 → **35** (унифицировано)
  - JUMP_FORCE_Y: 25 → **18**
- **Эффект:** Резкий, не парящий прыжок. Быстрое возвращение на землю.

#### 4. ✅ FORGIVENESS в Хитбоксах  
- **Файл:** `core/physics/CollisionSystem.ts`
