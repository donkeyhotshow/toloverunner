# Walkthrough: ToLOVERunner v2.4.0 - Combat & Interactive Comic

## Опис оновлення (Етап 5)

---

## 1. Бойова система (Combat v2.4.0)

### Jump Attack (UP + DOWN)
- Опис: Спеціальна атака для знищення ворогів типу COMBAT
- Управління: Свайп вгору (стрибок), потім свайп вниз
- Результат: Знищення ворога типу COMBAT

### Combo System (Streak)
- Лічильник комбо-серій
- Кожне влучне вбивство підвищує множник очок
- Промах або зіткнення скидають серію

### Combo UI
- Компонент: ComboDisplay
- Активація: При серії от 2 ударів
- Таймер: Відлік часу утримання комбо

---

## 2. Візуальні ефекти (Comic Visuals)

### Speed Lines
- Активація: Dash або високе комбо
- Ефект: 3D лінії швидкості

### Dutch Angle
- Динамічний нахил камери при атаках
- Сильніший нахил при комбо-майлстоунах

### Combat Particles
- Оновлена система частинок
- Яскраві та швидкі спалахи

---

## 3. Перешкоди та типи об'єктів

| Тип | Опис | Спосіб проходження |
|-----|------|--------------------|
| LOW | Перешкода на низькому рівні | Перестрибнути |
| HIGH | Перешкода на високому рівні | Проковзнути |
| COMBAT | Ворог | Jump Attack |

Технічна реалізація:
- Всі типи в ObjectType
- Обробка в CollisionSystem
