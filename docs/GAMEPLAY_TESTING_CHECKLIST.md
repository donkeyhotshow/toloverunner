### 10. ПРОИЗВОДИТЕЛЬНОСТЬ

#### 10.1 FPS (Frames Per Second)
Откройте консоль и выполните:
```javascript
setInterval(() => console.log('FPS:', window.__TOLOVERUNNER_PROFILER__.getMetrics().fps), 1000)
```

**Целевые показатели**:
- [ ] Desktop (средний ПК): FPS ≥ 55
- [ ] Desktop (слабый ПК): FPS ≥ 40
- [ ] Mobile: FPS ≥ 30

#### 10.2 System Timings
Выполните в консоли:
```javascript
window.__TOLOVERUNNER_PROFILER__.getSystemTimings()
```

**Ожидаемые значения**:
- [ ] `physics`: < 3 ms ✅ (< 5 ms допустимо)
- [ ] `particles`: < 2 ms ✅ (< 5 ms допустимо)

**Если > 5 ms** - красный флаг! 🚩

#### 10.3 Dynamic Culling (проверка адаптивности)
1. Откройте DevTools → Performance
2. Включите CPU throttling (4x slowdown)
3. Играйте 10 секунд
4. Проверьте консоль на сообщения:
   ```
   [DynamicCulling] Adjusted for FPS XX.X: { drawDistance: YY }
   ```

- [ ] При низком FPS drawDistance уменьшается (< 70)
- [ ] При высоком FPS drawDistance увеличивается (> 70)

#### 10.4 Плавность
- [ ] Нет "stuttering" (заикания)
- [ ] Нет "frame drops" (провалы FPS)
- [ ] Плавная анимация на протяжении 60+ секунд игры
