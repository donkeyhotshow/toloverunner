# ТЕХНИЧЕСКИЙ РЕГЛАМЕНТ И ЧЕК-ЛИСТ ПРИЕМКИ: SPERM RUNNER

Этот документ является финальным критерием готовности (Definition of Done). Любое отклонение от пунктов чек-листа классифицируется как баг (P1 или P2).

## 1. Модель персонажа и анимация
**Цель:** Достижение органического, «живого» поведения меша в условиях высокоскоростного движения.

| Критерий | Техническое требование | Статус |
| :--- | :--- | :--- |
| **Squash & Stretch** | Деформация по оси Y: `1.5x` при прыжке, `0.6x` при приземлении. Объем меша константен. | [ ] |
| **Nucleus (Ядро)** | Позиция ядра ограничена объемом головы. Использование Rim Light для читаемости через прозрачный меш. | [ ] |
| **Tail Physics** | Инерция хвоста (отставание `0.2–0.4с`). Плавность обеспечивается Triple Sine Wave или Verlet-интеграцией. | [ ] |
| **Прозрачность** | Материал головы (Alpha ~0.7). Корректная сортировка глубины (Depth Sorting) для ядра. | [ ] |

## 2. Физика и перемещение
**Цель:** Исключение проваливания сквозь геометрию и обеспечение предсказуемого управления.

- [ ] **Grounding:** Удержание над дорогой на высоте `0.05 units` через Raycast. Обновление позиции в `FixedUpdate` (для Unity) или через `requestAnimationFrame` с фиксированным шагом (для Three.js).
- [ ] **Гравитация:** Свободное падение `9.81 м/с²` при отсутствии детекции пола. Мягкая интерполяция при касании (avoid "snapping").
- [ ] **Lane Switching:** Смена полос за `0.2с` с кривой Ease-out. Блокировка ввода до завершения маневра (опционально).
- [ ] **Дрейф скорости:** Константное движение по оси Z. Ускорение только через внешние модификаторы (Dash/Boost).

## 3. Рендеринг дороги и окружения
**Цель:** Создание иллюзии бесконечного органического туннеля без визуальных швов.

- [ ] **UV Mapping:** Боковые грани дороги используют World-space или Planar маппинг. Растяжение текстур запрещено.
- [ ] **Бесшовность:** Перекрытие сегментов дороги на `0.01–0.05 units` для компенсации ошибок точности float.
- [ ] **Туман (Fog):** Тип: Exponential Squared. Цвет: `#500000`. Полное скрытие края отрисовки (Far Clip Plane).
- [ ] **Стены вены:** Вертексная анимация пульсации. Backface Culling настроен так, чтобы внутренние части меша были видимы.
- [ ] **Shadows:** Динамическая Blob-тень. Уменьшение Opacity и увеличение Scale при наборе высоты (прыжке).

## 4. Столкновения и «Juice» (Обратная связь)
**Цель:** Максимальный эмоциональный отклик на действия игрока.

- [ ] **Hit-box:** Коллизия регистрируется только по сфере головы. Хвост — визуальный элемент, не влияющий на поражение.
- [ ] **Hit Stop:** Заморозка мира на `50–80 мс` при столкновении. Шейдеры (пульсация) также должны замирать.
- [ ] **Camera Shake:** Тряска по трем осям. Интенсивность затухает по экспоненте.
- [ ] **VFX (Juice):**
    - Вспышка (Bloom) при подборе бонусов.
    - Частицы (планктон) дрейфуют навстречу игроку, создавая ощущение объема жидкости.
- [ ] **Ономотопея:** Появление текстов (SPLAT, BOING) через Billboard-систему (текст всегда смотрит в камеру).

## 5. Производительность и Оптимизация
**Цель:** Стабильная работа на мобильных устройствах среднего сегмента.

| Метрика | Целевое значение | Примечание |
| :--- | :--- | :--- |
| **FPS** | 60 (Min: 45) | Стабильность без фризов при генерации чанков. |
| **Draw Calls** | < 150 | Использование GPU Instancing для препятствий. |
| **Memory Leak** | 0 MB / 10 min | Обязательный Object Pooling для сегментов дороги. |
| **Z-Fighting** | Отсутствует | Смещение мешей по оси Y на `0.001` для теней и декалей. |
