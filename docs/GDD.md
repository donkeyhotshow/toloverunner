## 5. ПОЛОСЫ ДВИЖЕНИЯ (LANES)

> Источник: `utils/laneUtils.ts`, `constants/gameplay.ts → LANE_WIDTH`, `constants/world.ts → SAFETY_CONFIG`

- **5 полос** с индексами: `−2, −1, 0, +1, +2`
- **Ширина полосы:** 2.0 единицы
- Игрок начинает на центральной полосе `0`
- Перемещение: мгновенная смена целевой полосы → пружинная интерполяция (stiffness 280, damping 18)
- Коэффициент скорости перехода: `28 units/s` (laneSpeed в PlayerPhysics)

```
Полосы X:  -4.0  -2.0   0    +2.0  +4.0
Индексы:    -2    -1    0     +1    +2
```

## 6. СКОРОСТЬ И ПРОГРЕССИЯ

> Источник: `constants/gameplay.ts → GAMEPLAY_CONFIG, RUN_SPEED_BASE`, `store/gameplay/speedActions.ts`

| Параметр | Значение |
|----------|---------|
| Начальная скорость | 10.0 ед./с |
| Минимальная скорость (в игре) | 12.0 ед./с |
| Максимальная скорость | 45.0 ед./с |
| Прирост скорости за монету | +0.012 ед./с |
| Прирост скорости за дистанцию | +0.5 ед./с каждые 500 м |
| Speed Boost powerup | +15 ед./с на 5 секунд |
| Замедление после удара | ×0.75 от текущей скорости |

**Формула прироста скорости за монету** (fixed-point, без дрейфа):
```
newSpeed = min(MAX_SPEED, (round(speed × 1000) + 12) / 1000)
```
Скорость монотонно растёт и фиксируется на 45 — никогда не превышает.

## 7. ОЧКИ И СИСТЕМА КОМБО

> Источник: `store/gameplay/speedActions.ts`, `store/gameplay/comboActions.ts`

### 7.1 Источники очков

| Событие | Очки |
|---------|-----|
| 1 м дистанции | +1 |
| Монета (базово) | +5 |
| Монета (perfect timing, <500 мс после прошлой) | +5 + 50 (бонус) |
| Монета (с учётом множителя) | `(5 + bonus) × multiplier` |
| Graze (пролёт рядом с врагом) | +50 |
| Прыжок через бактерию | +10 |
| Сбор гена DNA | +500 |
| Максимум очков | 9 999 999 |

### 7.2 Комбо-множитель

- Каждая монета / graze увеличивает `combo` на 1
- **Множитель** = `floor(combo / 5) + 1` — растёт каждые 5 монет
- Таймер комбо: 3.5 с — если не собирать монеты, комбо сбрасывается
- При получении урона: `combo = 0`, `multiplier = 1`

### 7.3 Геймплейная валюта — гены (`genesCollected`)

- За каждую монету: +5 генов
- Тратятся в магазине на апгрейды
- Сохраняются между сессиями (localStorage)

## 8. ЖИЗНИ, СМЕРТЬ И ВОЗРОЖДЕНИЕ

> Источник: `constants/gameplay.ts → INITIAL_LIVES`, `store/gameplay/damageActions.ts`, `store/sessionSlice.ts`

| Параметр | Значение |
|----------|---------|
| Стартовые жизни | 1 ❤️ |
| Макс. жизней (начально) | 1 |
| Макс. жизней (с апгрейдом) | Неограничено (покупается в магазине) |
| Задержка Game Over после смерти | 1.0 с |

### 8.1 Получение урона

1. Попадание в **вирус** → мгновенная смерть (обходит щит и неуязвимость)
2. Попадание в **препятствие / врага** (не вирус):
   - Если активен **щит** → щит разрушается (1 заряд)
   - Если нет щита, нет неуязвимости → `lives -= 1`, скорость ×0.75, неуязвимость 2.5 с
   - `lives = 0` → `deathTimer = 1.0` → `GAME_OVER`

### 8.2 Возрождение

- Кнопка **REVIVE** появляется на экране Game Over только если `gems ≥ 1`
- Стоимость: 1 💎 (gem)
- Эффект: `lives = 1`, статус → `PLAYING`

## 14. БОЕВАЯ СИСТЕМА (COMBAT)

> Источник: `store/gameplay/comboActions.ts → setAttack`, `components/Gameplay/Combat/useCombatSystem`

При прыжке / скольжении активируются атаки:
- **Прыжок** (`Space`/`↑`) → атака `UP` — уничтожает врагов сверху
- **Скольжение** (`S`/`↓`) → атака `DOWN` — атакует снизу
- Таймер атаки: 0.4 с
- Типы врагов `COMBAT` (в `ObjectType`) уничтожаются боевыми атаками

## 15. РЫВОК (DASH)

> Источник: `store/gameplay/damageActions.ts → dash`, `constants/physicsConfig.ts → PLAYER_PHYSICS`

- Клавиша: `Shift`
- Даёт неуязвимость на **0.5 с** (длительность дольше самого рывка: 0.35 с)
- Кулдаун: **2.0 с**
- Dash chain: если второй рывок в течение 1 секунды — счётчик цепочки растёт
- При рывке активируется `isDashing = true`, затем `isInvincible = true`

## 19. ВИЗУАЛЬНЫЙ СТИЛЬ И 3D

### 19.1 Арт-стиль

- **Comic / Pop-Art** — жирные чёрные контуры (border-black), halftone паттерны
- **Шрифт заголовков:** Bangers (pop-comic)
- **Toon-shading:** `MeshToonMaterial` для 3D объектов
- **Post-processing:**
  - Bloom (свечение)
  - Chromatic Aberration (хроматическая аберрация)
  - Vignette (виньетка с пульсацией)
  - Speed Lines (при высокой скорости / комбо ≥ 10)
- **Tone mapping:** `ACESFilmicToneMapping` (кинематографический)

### 19.2 3D-сцена

| Элемент | Описание |
|---------|----------|
| Фон сцены | `#1a0515` — тёмный органический фиолет |
| Туман | `FogExp2('#1a0515', 0.001)` — экспоненциальный |
| Освещение | Ambient `intensity 1.5` + Directional `[0,10,5] intensity 2` |
| Камера | `position [0, 3, 8]`, `fov 70`, `far 3000` |
| Антиалиасинг | Включён (`antialias: true`) |
| Точность | `highp`, log depth buffer |
| DPR | `[1, 2]` (адаптивно) |

### 19.3 Финальный объект — Яйцеклетка

> Источник: `components/World/FinishEgg.tsx`

- Сфера радиуса **15 единиц** (огромная, видна издалека)
- Материал: `MeshToonMaterial`, золотистый / оранжевый градиент
- Наружное свечение (`BackSide mesh`)
- Появляется на дистанции `WIN_DISTANCE = 3000 м`
- При достижении → `VICTORY`
