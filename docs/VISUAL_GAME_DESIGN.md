## 1. МОДЕЛЬ ИГРОКА И АНИМАЦИИ

### 1.1 Внешний вид персонажа

Персонаж — стилизованный **сперматозоид** в минималистичном 3D-исполнении.

**Версия v2.2.0 (мобильный, 2D-сайд-скроллер):**
- **Голова:** квадратная/блочная, ярко-розовый/малиновый (#E84A6F), крупная, занимает ~40% тела
- **Хвост:** бирюзово-голубой (#4ECFCF), вытянутый, волнообразный
- **Акцент:** жёлтый/золотой ромб (#FFD700) спереди — «нос»

**Версия v2.4+ (текущий 3D):**

На скриншотах (content.png, content (1).png — (3).png) и referral-скриншоте (content (7).png) виден персонаж:
- **Форма:** вытянутое **каплевидное тело** с загнутым хвостом, белое/светло-серое
- **Голова:** круглая, крупная (bulb), белая с лёгкими деталями (иконка лица или пустая)
- **Хвост:** тонкий изогнутый хвост, уходящий влево-вниз
- **Стиль:** чистый минимализм, органическая форма без «комиксного» контура
- **Пропорции:** голова ≈ 1/3, тело+хвост ≈ 2/3

```
     ●
    / ← голова (большой круг/bulb)
   |
    \___  ← изогнутый тонкий хвост
```

### 1.2 Цветовая схема персонажа

| Элемент | v2.2 (mobile) | v2.4+ (3D референс, content(7)) |
|---|---|---|
| Тело/голова | Розовый `#E84A6F` | Белый `#F0E8E0` |
| Хвост | Бирюзовый `#4ECFCF` | Белый/светло-серый |
| Акцент | Жёлтый ромб `#FFD700` | — |
| Тень/контур | Чёрный `#000000` | Мягкая тёмная тень |

> **Решение для v3.0.0:** Персонаж должен остаться белым/светлым — это обеспечивает максимальный контраст на тёмных фиолетовых туннелях (биомы Fallopian Tube, Sperm Duct). На светлых биомах (Egg Zone) возможна лёгкая тёплая тонировка.

### 1.3 Анимации

| Состояние | Описание | Визуальный сигнал |
|---|---|---|
| **Бег** | Тело плавно движется вперёд, хвост синусоидально колышется | Волна хвоста ~2 цикла/сек |
| **Прыжок** | Дуга вверх, хвост вытягивается, приземление с "пружинкой" | Стикер BOING! |
| **Двойной прыжок** | Второй импульс в воздухе | Короткий flash вокруг персонажа |
| **Скольжение** | Тело прижимается к земле, хвост горизонтально | Тело уменьшается по высоте |
| **Dash** | Резкое ускорение, след за хвостом (motion blur) | Иконка ⚡ активируется |
| **Смерть** | Столкновение → flash → заморозка → WASTED экран | Экран краснеет |
| **Старт (из 3)** | Цифра "3" трансформируется в персонажа | Морф анимация |
| **Perfect** | Прохождение точно в кольцо/зону → gold glow | "PERFECT TIMING!" / "PERFECT ZONE!" |
| **Idle** | Лёгкое покачивание на месте (в меню) | Медленная волна хвоста |

## 2. ДОРОГА И ТУННЕЛЬ

### 2.1 Туннель (главная сцена)

**Главный визуальный элемент игры** — бесконечный **круглый туннель** с процедурно генерируемыми стенами.

Из скриншотов (content.png — content(6).png):

**Геометрия туннеля:**
- Форма: идеально круглый цилиндр, camera смотрит вдоль оси
- Стены: состоят из концентрических рёберных колец (ribbed texture), создающих ощущение глубины
- Перспектива: центральная, создаёт 3D иллюзию движения вглубь

**Биомы туннеля (задокументировано по скриншотам):**

| Биом | Цвет стен | Цвет рельс/пола | Атмосфера |
|---|---|---|---|
| **Sperm Duct / Fallopian (purple)** | Насыщенный фиолетовый `#6A0DAD` | Фиолетовые полосы | Пульсирующий, deep |
| **Vein Tunnel (тёмный)** | Тёмно-фиолетовый `#2D0D4D` | Тёмные рёбра | Клаустрофобный |
| **Fallopian Tube (тёплый)** | Розово-телесный `#D4856A` | Оранжево-коричневые рельсы | Органический, мягкий |
| **Vein Tunnel / Ovary (красный)** 🆕 | Ярко-красный `#CC1A1A`, розово-красные блобы | Белая/светлая дорожка, коричневые рельсы | Горячий, органический, агрессивный |

**Детали стен (content(7).png — лучший референс Fallopian биома):**
- Стены покрыты органическими **наростами/ворсинками** — как реальная маточная труба
- Текстура: складчатая, мягкая, тёплые розово-телесные тона
- На фоне в конце туннеля виден **круглый портал** (тёмный с звёздами) — визуальная цель

**Детали красного биома (новые скриншоты):**
- Стены: насыщенный красно-розовый `#CC1A1A`, текстура мягкая (не рёбра, а органические пятна)
- По бокам высокая **зелёная растительность** — тростник/водоросли `#3A8A3A`, высота ~1.5 unit
- Фоновые красные блобы — парят по сторонам дорожки (мягкие, округлые пятна `#E03030`)
- В самой глубине — **тёмно-синий круг** с белыми звёздными частицами — выглядит как космический портал
- Над порталом — **белая вертикальная полоска** (целевой объект / obstacle ahead)

**Рельсы/дорожка (красный биом — детально):**
- Дорожка (floor): **белая / светло-серая** прямая полоса, уходящая в перспективу
- Рельсы: **коричнево-деревянные** брёвна/перила по обеим сторонам, диагонально уходят вглубь
- Ширина дорожки: ~3 unit (3 полосы), строгий центральный путь
- Текстура пола: без явных полос — однородный светлый (контраст с тёмным у фиолетового биома)

### 2.2 Фон и атмосфера

**Дальний план туннеля (единая система у всех биомов):**
- В глубине всегда виден **тёмно-синий/чёрный круг** — как зрачок туннеля
- Внутри круга: белые/золотые звёздные частицы, слабое свечение
- Он создаёт иллюзию бесконечности и является визуальным горизонтом
- При смене биома фон плавно crossfade-ится (2–3 сек)

**Частицы в туннеле:**
- Красные частицы/дебрис разлетаются по сторонам (видно на content(1).png)
- Белые/бирюзовые монеты при COIN RAIN
- Белые глитч-частицы в красном биоме (атмосфера)
- Генерируются через `core/particles` с бюджетом ≤ 50 единиц

## 3. ОБЪЕКТЫ В ИГРЕ

### 3.1 Кольца (Rings / Yellow Hoops)

**Двух видов, видны на всех gameplay скриншотах:**

**Тип A — Жёлтое 3D кольцо (прохождение):**
- Форма: тор (donut shape), жёлтый насыщенный `#FFD700`
- Поверхность: объёмная, пятнистая органическая текстура (cracked/spotted pattern — как яйцо)
- Размер: крупный, перекрывает ~2/3 ширины дорожки
- Позиционирование: стоит на дорожке вертикально, центрировано по полосе
- **PERFECT TIMING!** → баннер +100 Bonus Points при точном прохождении через центр
- **PERFECT ZONE!** → серия колец — жёлтый гриббон `+100` появляется после каждого
- На новых скриншотах: вид близкого кольца (крупный план) — явно 3D torus с тёплой пятнистой текстурой

**Тип B — Малое кольцо / outline кольцо (сбор):**
- Меньше Типа A примерно в 2–3 раза
- Тёмный/контурный вид — чёрная обводка, без заливки
- Летит навстречу или расположено ближе к земле
- Действие: подобрать (без прыжка) или уворот

### 3.2 Тёмные враги / Globus-типы

На скриншоте content.png и content(1).png справа от игрока видны **тёмные чёрные органические блобы:**
- Форма: неправильная, "кляксообразная", угловатые выросты
- Цвет: чёрный / тёмно-серый
- Стиль: матовый, без блеска — выглядят как бактерии или черви
- Это: `GLOBUS_*` или `BACTERIA_*` типы врагов
- Действие против них: перепрыгнуть, уклониться, атака вниз/вверх

### 3.3 Препятствие "Obstacle Wave" 

**Content (4).png** — чёткий референс:
- Объект: вертикальная **стойка** с перекладиной → **знак T-образный**
- Верх стойки: круглая голубая/тёмная голова
- Тело: тонкая вертикальная белая/светлая полоска
- При приближении появляется баннер: **⚠️ OBSTACLE WAVE!** (зелёный пузырь с жёлтым треугольником)
- Действие: **слайд под перекладиной**

### 3.4 Баннер Speed Boost

**Content (2).png:**
- Баннер: зелёный закруглённый прямоугольник, белый текст **"🚀 SPEED BOOST!"**
- Появляется при подборе power-up
- Позиционируется над или рядом с персонажем (speech bubble стиль)

### 3.5 Монеты / Collectibles

- Жёлтые объекты падают вертикально при COIN RAIN (content.png — золотые точки снизу)
- Монеты притягиваются при активном Magnet power-up
- Гены (Genes) — коллекционная валюта для DNA-карт

### 3.6 Красные летящие объекты

На нескольких скриншотах (content.png, content(1).png) по бокам туннеля летят **красные осколки/дебрис:**
- Вытянутые, красные `#CC2020`, хаотичное вращение
- Атмосферный эффект (не interactable) или осколки разбитых врагов

**SCORE (верхний левый):**
- Иконка монеты (💰) + зелёный индикатор `+200` при сборе
- Число: белый жирный шрифт на тёмном фоне
- При комбо показывает `+100`, `+200` pop-up анимацией

**Дистанция (под score):**
- Бирюзовая метка `55m / 86m` — прямоугольный badge, цвет `#4ECFCF`

**DASH кнопка (левый край):**
- Белая карточка `DASH [S]` с чёрным текстом
- Внутри: иконка молнии ⚡ (чёрная на белом)
- При кулдауне: иконка затемняется, граница серая
- При активации: lightning burst анимация

**COMBO карточка (правый край):**
- Жёлтая карточка, чёрный текст `COMBO`
- Силуэт персонажа (чёрный) + красный круг с числом множителя `x2`
- Иконка огня 🔥 рядом с множителем
- При высоком комбо (5+): мигает/анимируется

**Жизни (Hearts) — верхний центр:**
- Розовые/красные сердечки ❤️, пустые ○ при потере жизни
- 3–4 сердечка в ряд

**Таймер (верхний правый):**
- Иконка секундомера ⏱ + время в формате `00:01`
- Считает вверх (сколько бежишь)

**Session timer (нижний левый):**
- Маленький счётчик `00:10 / 00:27` — суммарное время сессии

## 5. ДИЗАЙН-ДИРЕКТИВЫ ДЛЯ v3.0.0

### 5.1 Что сохранить из v2.2 (сильные стороны)

- ✅ Comic-стикеры при событиях (BOING!, POW!) — их нужно оставить в 3D версии как 2D billboard overlay
- ✅ Чёрный/тёмный portal в глубине туннеля — создаёт ощущение цели
- ✅ Чистый белый персонаж с изогнутым хвостом — хорошая читаемость
- ✅ Жёлтые кольца — мгновенно читаются как "цель для прыжка"
- ✅ Баннеры событий (speech bubble стиль) — информируют без отвлечения

### 5.2 Что улучшить в v3.0.0

- 🔧 **Туннель:** добавить анимированные стены (пульсация, перистальтика) для ощущения живого организма
- 🔧 **Персонаж:** добавить тонкие particle-эффекты вокруг хвоста при беге (trail)
- 🔧 **Кольца:** сделать кольца слегка светящимися (emissive жёлтый, intensity 0.3) — видны издалека
- 🔧 **Биом-переход:** плавный crossfade за 2–3 сек через туман + ambient light
- 🔧 **HUD:** перейти на glassmorphism из rgba(10,0,20,0.75) + blur(12px)
- 🔧 **Враги:** добавить анимированный wobble-эффект для Globus типов (органическое колыхание)

### 5.3 Дизайн-запреты

- ❌ Неоновые цветные **персонаж** (нарушает читаемость на тёмных биомах)
- ❌ Прямоугольный туннель (нарушает органическую тематику)
- ❌ Реалистичные текстуры с высокой детализацией (рушит стиль + draw calls)
- ❌ Чистые RGB цвета в 3D геометрии (только вариации с desaturation)

## 7. ПЕРСОНАЖ — УТОЧНЕНИЕ (из новых скриншотов)

На новых 5 скриншотах (красный биом) **чётко виден персонаж в движении**:

- **Форма:** каплевидная голова (bulb) + тонкий изогнутый хвост уходит **вниз-вправо** (как реальный сперматозоид)
- **Цвет:** матово-белый `#F0EEE8`, без блеска, мягкая тень снизу
- **Хвост:** хаотично изогнут в движении, видно нескольких кадров (анимация волны)
- **Тень:** мягкая тёмная тень на полу под персонажем (real-time shadow)
- **При Dash:** за хвостом появляется бирюзовый световой след (`#4ECFCF`)
- **Голова (bulb) детали:** слегка вытянутая вверх, не идеально круглая — ближе к egg-shape
- **Шея/соединение:** тонкое место перехода головы в хвост — изящный изгиб

## 8. АНАЛИЗ КОНЦЕПТОВ  ЦЕЛЕВОЙ ДИЗАЙН

### 8.1 Туннель — целевая визуализация (концепты 1, 2)

Стены «биосферы» — населены микроорганизмами (грибы, медузы, ДНК, бактерии):
- Цвет стен: складчатый розово-лососёвый — органическая ткань фаллопиева канала
- Portal arch: хрящи/кости + кристаллы + aurora + звёзды  внутри тёмного круга
- Пол: розово-лавандовая плиточная текстура
- Рельсы: тёмные ж/д рельсы (а не брёвна)
- Монеты: жёлтые кружки с ДНК-спиралью внутри
- DNA-сфера: голубой орб с ДНК  коллектибл для DNA-карт
- Shop Portal: фиолетовая шайба на полу с glow-эффектом

### 8.2 Персонаж  эталон

White sperm: крупная белая яйцевидная голова (bulb) + тонкий волнообразный хвост.
При Dash: speed-bryzgi + стикер ZOOM!

Y-хромосомный персонаж (реальный 3D, скриншот 3): чёрный bulb + крылья + жёлтое кольцо.

### 8.4 Уровни  два типа

- Туннель (основной): бег вглубь трубы, органические рёберные стены, 3 полосы
- Открытый мир (концепт 4): дорога по ландшафту с ДНК-спиралями, финальная цель  Boss-яйцо

### 8.5 Мультиплеер  Race концепт (последний скриншот)

Текст: Кто первый добежит  определяет пол!
- 3 персонажа плывут одновременно по извилистой дороге к яйцу
- Персонажи отличаются аксессуарами: ковбойская шляпа, цилиндр, пропеллер-шляпа
- Финальная цель: Яйцо-королева с короной  Boss на горизонте
- HUD: прогресс-бар у каждого игрока снизу, монеты, молния (Dash)
- Это режим RACE / COOP из GameMode enum

### 8.6 Финальная цель  Яйцо

- Яйцо-королева (Egg Queen): крупный персонаж с короной, улыбается
- Окружена биологическим ландшафтом (ДНК-спирали, тростник, существа)
- Это финал 3000 м  VICTORY экран
- Визуально: бежевое яйцо, выражение лица  дружелюбное/нейтральное/удивлённое
